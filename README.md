# Inference

A weekly competition for people who like thinking about uncertainty. Three problems, three weeks, no code required.

Built with Next.js, Supabase, and Tailwind. Deployed on Vercel.

## Development

```bash
pnpm install
pnpm dev
```

Runs at [localhost:3000](http://localhost:3000).
